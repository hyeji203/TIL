2. 정사각형 격자 + X자 출력
void main() {
  var size = 7;
  for (var i = 0; i < size; i++) {
    printSharp(size, i);
  }
}

void printSharp(int size, int y) {
  var result = "";
  for (var x = 0; x < size; x++) {
    var condition =
        x == y ||                  // ↘ 대각선
        x + y == size - 1 ||       // ↙ 대각선
        x == 0 || x == size - 1 || // 좌우 테두리
        y == 0 || y == size - 1;   // 상하 테두리
    if (condition) {
      result += '#';
    } else {
      result += ' ';
    }
  }
  print(result);
}

- (x, y) 좌표를 기준으로 조건을 세워 # 또는 공백을 출력한다
- 조건을 조합하면 테두리와 X자를 동시에 표현할 수 있다

3. ^ 모양을 여러 번 출력
void main() {
  int n = 3;      // 삼각형 높이
  int count = n;  // 반복 횟수

  for (int y = 0; y < n; y++) {
    String result = '';
    for (int c = 0; c < count; c++) {
      for (int x = 0; x < n * 2 - 1; x++) {
        if (x == n - 1 - y || x == n - 1 + y) {
          result += '#';
        } else {
          result += ' ';
        }
      }
      result += ' '; // 삼각형 사이 간격
    }
    print(result);
  }
}

- (n*2 - 1) 길이의 문자열에서 x == n-1-y 또는 x == n-1+y일 때 # 출력
- 이를 count번 반복해서 가로로 여러 개의 ^ 모양을 붙인다

4. ^ 모양을 여러 번 출력2
void main() {
  var n = 5;

  for (var y = 0; y < n; y++) {
    var line = '';
    for (var x = 0; x < (n * 2 - 1) * n; x++) {
      var condition = (x % (n * 2 -1)) - (n - 1) == y ||
                      n - (x % (n * 2 -1)) - 1 == y;
      if (condition) {
        line += '#';
      } else {
        line += ' ';
      }
    }
    print(line);
  }
}

- % 연산을 활용해 반복되는 패턴을 한 줄로 처리
- (x % (n*2 - 1)) → 삼각형 하나의 좌표계로 환원

✨ 오늘 배운 점
- 단순 print → 작은 패턴에만 적합
- 반복문 + 조건문 → 좌표 기반 사고로 다양한 도형을 표현 가능
- % 연산 → 패턴을 반복적으로 이어붙이는 데 유용
