## 1. 핫 리로드(Hot Reload)란?
- 실행 중인 앱을 완전히 재시작하지 않고, 수정한 Dart 코드만 가상 머신(VM)에 주입해서 UI를 즉시 업데이트하는 기능
- 앱을 껐다 켜지 않아도 화면이 바로 반영되기 때문에 개발 속도가 엄청 빨라짐

## 2. 3가지 기본 위젯(Widget)
### 1) **StatelessWidget**
- 불변 위젯 - 상태가 변하지 않음
```dart
class MyApp extends StatelessWidget { ... }
```
### 2) **StatefulWidget**
- 가변 위젯 - 상태 변경 가능
```dart
class MyHomePage extends StatefulWidget { ... }
```
### 3) **InheritedWidget**
- 위젯 트리에서 데이터 공유
- Provider의 기반 기술
- 불필요한 리빌드 방지
