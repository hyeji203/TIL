#  파이썬 반복문  복습 노트

## 1. 반복문이란?
* **정의:** 동일한 문장을 여러 번 반복시키는 구조 (Iteration)
* **중요성:**
    * 컴퓨터는 반복적인 작업을 실수 없이 빠르게 수행할 수 있음 (컴퓨터의 가장 큰 장점)
    * 코드의 복사-붙여넣기를 방지하고 효율적으로 작성 가능
* **종류:**
    1.  **횟수 제어 반복 (`for` 문):** 정해진 횟수만큼 반복
    2.  **조건 제어 반복 (`while` 문):** 특정한 조건을 만족하는 동안 계속 반복

---

## 2. 횟수 제어 반복: `for` 문

### 2.1 기본 구조
시퀀스(리스트 등)의 항목들을 하나씩 가져와서 반복합니다.
```python
for i in [1, 2, 3, 4, 5]:   # 끝에 콜론(:) 필수
    print("방문을 환영합니다.") # 들여쓰기 필수
````

### 2.2 `range()` 함수 활용

반복 횟수를 생성할 때 주로 사용합니다.

  * `range(stop)`: 0부터 stop-1까지
  * `range(start, stop)`: start부터 stop-1까지
  * `range(start, stop, step)`: start부터 stop-1까지 step 간격으로

**예제:**

```python
# 1부터 5까지 출력
for i in range(1, 6, 1):
    print(i, end=" ")
```

## 3\. 조건 제어 반복: `while` 문

### 3.1 기본 구조

조건식이 `True`인 동안 블록 내의 문장을 계속 실행합니다.

```python
password = ""
while password != "pythonisfun":
    password = input("암호를 입력하시오: ")
print("로그인 성공")
```

### 3.2 무한 루프와 `break`

  * **무한 루프:** 끝없이 반복되는 구조 (예: `while True:`)
  * **break:** 루프를 강제로 빠져나올 때 사용

<!-- end list -->

```python
while True:
    light = input('신호등 색상: ')
    if light == 'blue':
        break  # 루프 탈출
    print('전진!!')
```

## 4\. 주요 실습 예제 (Labs)

수업 시간에 다룬 주요 실습 코드의 로직입니다.

### 🐢 터틀 그래픽 활용

1.  **n각형 그리기:** `range(n)`을 사용하여 `360/n`도씩 회전하며 선 그리기
2.  **별 그리기:** 5번 반복하며 144도씩 회전
3.  **랜덤 이동:** `random` 모듈을 사용하여 거북이의 이동 거리와 각도를 난수로 설정
4.  **스파이럴(나선):** 반복할 때마다 선의 길이를 조금씩 늘리고 각도를 약간 비틀어(89도) 그리기

### 🧮 계산 및 응용

1.  **팩토리얼 계산:** 1부터 n까지의 수를 반복해서 곱함 (`fact = fact * i`)
2.  **구구단 출력:** 사용자에게 단을 입력받아 1\~9까지 곱셈 결과 출력
3.  **숫자 맞추기 게임:**
      * 컴퓨터가 1\~100 사이 난수 생성
      * `while` 문으로 사용자가 맞출 때까지 반복 (Up/Down 힌트 제공)
4.  **모든 샌드위치 조합:**
      * 중첩 반복문(Nested Loop) 사용
      * 빵, 고기, 야채, 소스 리스트를 각각 반복하여 모든 경우의 수 출력
5.  **암호 해독 (무차별 대입):** 3중 반복문으로 모든 알파벳 조합을 생성하여 비밀번호와 비교
